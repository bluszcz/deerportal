Detailed Description

The BoardDiamondSeq class We are going to have 2 cards / diamonds of each element per each area, which gives 2*4*4 = 32 cards diamonds of elements on the board.

Additionally there will be extra 6 white diamonds per area to collect, which gives 4*6 = 24 diamonds on the board.

Together it would give 32 + 24 = 56 diamonds cards/diamonds together, 15 per area.
